# liquityWidget
liquityWidget은 BOS를 활용한 Liquity의 분산형 dApp입니다.

[liquityWidget](https://near.org/0xgh.near/widget/liquityWidget) 애플리케이션은 현재 Sepolia Testnet에서만 실행해 볼 수 있으며, 곧 Etherium Mainnet에서도 사용할 수 있게 업데이트 할 예정입니다.
